## Setup

1. Ensure your device is on the same WiFi network as your Bose SoundTouch speaker
2. Find your Bose device's IP address (check your router or the old Bose app)
3. Enter the IP address in the app
4. Start controlling your speaker!

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Lucide React Icons
- Bose SoundTouch Web API
